Pin Your blog post Images on image hover.

== Description ==

Dynamic Pin It Button On Image Hover plugin use the Pinterest javascript library to pin your pictures on your WordPress site and the person is required to log in to Pinterest for the photo to be pinned to your Pinterest account.

**Features**

* Show Pinterest Pin It Button On Image Hover
* Show Pinterest Pin It Button In Post
* Show Pinterest Pin It Button In Post Archive
* Pinterest Pin It Button Settings
* Customizable Button Size On Hover
* Multilingual - Translatable in Any Language

== Installation ==

1. Upload `dynamic-pinterest-pin-it-button-on-image-hover.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
